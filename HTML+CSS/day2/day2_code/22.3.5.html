1. css引入方式
* 内部样式 使用style标签设置css样式
* 外部样式
* 使用link标签的href属性引入外部css文件
* 使用style标签的@import引入外部css文件
* 行内样式 style作为属性直接卸载标签后面
2. 三种引入方式优先级比较
!important>行内>内部>外部
3. 选择器
* 元素选择器
* 类选择器
* id选择器
* 通配符选择器
* 群组选择器和后代选择器
* 伪类选择器 必须遵循以下顺序设置
a:link{属性：属性值} 超链接的初始状态
a:visited{属性：属性值} 超链接被访问后的状态
a:hover{属性：属性值} 鼠标悬停时的状态
a:active{属性：属性值} 超链接被激活时的状态
4. 选择器的权重
！important > 行内 > 包含选择符 > id > 类选择器 > 元素选择器
选择器解析规则：
* 当不同选择符的样式设置具有冲突时，高权重选择符的样式会覆盖低权重选择符的样式
* 相同权重的选择符，样式遵循就近原则；哪个选择符最后定义，就采用哪个选择符的样式
5. 文字颜色的取值
* 直接写颜色
* rgb(255，0，0)格式 red green blue的缩写
* #000000 十六进制写法 两位代表一个颜色 同样为红绿蓝
